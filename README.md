**Verilog Digital Design Library**

This repository contains Verilog implementation of basic logic gates and combinational circuits.  
It is useful for students learning Digital Electronics, VLSI, and Verilog HDL.

---

<u>**Project Structure**</u>

<u>**-> Basic Gates**</u>

1) NOT Gate  
2) AND Gate  
3) OR Gate  
4) NAND Gate  
5) NOR Gate  
6) XOR Gate  
7) XNOR Gate
8) Buffer Gate  

<u>**-> Combinational Circuits**</u>

1) Half Adder  
2) Full Adder  
3) Half Subtractor  
4) Full Subtractor  
5) Multiplexer  
6) Demultiplexer  
7) Encoder  
8) Decoder  
9) Priority Encoder  
10) Carry Look Ahead Adder (CLA)  
11) Tri State Buffer  

---

<u>**Tools Used:**</u>

1) Verilog HDL  
2) Icarus Verilog  
3) GTKWave  
4) VS Code  

---

<u>**Features:**</u>

- Clean and simple Verilog code  
- Beginner friendly implementations  
- Covers fundamental digital concepts  
- Includes multiple combinational designs  

---

<u>**Learning Outcomes:**</u>

- Understanding of basic logic gates  
- Design of combinational circuits  
- Writing Verilog modules and testbenches  
- Simulation and verification
